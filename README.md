### Static Analysis Tools
1. APKTool ‚Äì Decompile/modify APK (smali-level)
2. Jadx / JD-GUI ‚Äì Convert DEX to readable Java code
3. MobSF ‚Äì Automated static + dynamic scanner
4. Androguard ‚Äì Python tool for APK/DEX/smali analysis
5. Bytecode Viewer ‚Äì Reverse engineering with multiple decompilers
6. ClassyShark ‚Äì Explore APK classes/methods/manifest
7. Qark ‚Äì Detects security issues in APKs
8. Enjarify / dex2jar ‚Äì DEX to Java JAR conversion
9. APKLeaks ‚Äì Extract secrets, tokens, and URLs

### Dynamic Analysis Tools
1. Frida ‚Äì Hook/modify functions at runtime
2. Objection ‚Äì Runtime exploitation via Frida (no root required)
3. Xposed / LSPosed ‚Äì Framework for modifying app behavior
4. Burp Suite ‚Äì Intercept/modify network traffic
5. Drozer ‚Äì Android app attack framework
6. Magisk ‚Äì Systemless root; works with LSPosed modules
7. adb ‚Äì Debugging bridge for Android device

### Logging Tools
1. Logcat ‚Äì Default Android logging system (adb logcat)
2. Pidcat ‚Äì Filtered Logcat output by package
3. MatLog ‚Äì GUI log reader (useful for non-rooted devices)
4. XLog / Timber ‚Äì In-app logging libraries used in apps
5. Logd ‚Äì Android logging daemon behind logcat
6. Syslog ‚Äì For rooted devices to log everything (system + kernel)
